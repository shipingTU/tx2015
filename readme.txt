Système recommandé : 
●	Linux Debien / Ubuntu


Logiciel requis : 
●	Apache + PHP5 (version >= 5.5)
○	MySQL avec phpmyadmin
●	Sendmail
○	Serveur FTP
●	serveur SSH


Fonctionnements du site :
●   Gestion des pages statiques de manière dynamique
○   Gestion des journaux spécifiques
●   Gestion des dossiers dans le "mini dropbox"
○   Gestion des messages via Contact
●   Gestion des utilisateurs (protéger des mots de passe en hachage) : admin, user interne et "visiteurs"


Très Important : Comme nous avons choisi d’utiliser uniquement le format UTF-8 pour une meilleur compatibilité, La base de données doivent être créé en format UTF-8, et le système des fichiers doit etre compatible avec UTF-8 (les noms des fichiers doivent etre encodes en UTF-8).
